# Pre-Celadon Type Opportunity Audit

## Scoring Formula
- +1.0 per obtainable species of that type (before arriving at Celadon).
- +0.1 per level-up move of that type learnable at level <= 30 by an obtainable species.
- +0.5 * map encounter_rate for each wild encounter area (land/water/rock/fishing) where the type appears on a pre-Celadon map.

## Pre-Celadon Scope Assumption
- Included Kanto progression through Lavender/Vermilion/Cerulean paths and adjacent approach routes (Route 7/8), excluding Celadon maps themselves.
